    Daten von der Uhr aktualisieren

    Der Button startet den GitHub-Workflow, der frische Daten aus Garmin Connect zieht und die Seite neu veröffentlicht (Dauer: ~2–3 Minuten).

    Dafür braucht es einmalig einen GitHub-Token (Fine-grained PAT, Repo GarminTool, Berechtigung „Actions: Read and write“). Token erstellen ↗

    Der Token wird nur lokal in diesem Browser gespeichert (localStorage) und ausschließlich an api.github.com gesendet.
